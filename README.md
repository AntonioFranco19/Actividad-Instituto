Actividad educativa enfocada en el desarrollo de clases para gestionar la información de un instituto llamado "TecnoFuturo". Describe la creación de una clase central llamada Centro que agrupa ciclos formativos y alumnos, y la refactorización del programa principal para trabajar con esta nueva arquitectura.

- Clase Centro como entidad principal: Se debe crear la clase Centro con propiedades de identificación, nombre, dirección y teléfono, además de listas para almacenar objetos CicloFormativo y Alumno. El constructor debe inicializar estas propiedades y listas para evitar errores de referencia nula.
- Refactorización del menú principal: El programa debe centrarse en una instancia de Centro, eliminando datos fijos iniciales. Se introduce una variable para el ciclo formativo seleccionado y se modifica el menú para crear, listar y seleccionar ciclos, mostrar detalles, agregar módulos, matricular alumnos y listar estos, siempre validando la selección actual.
- Resumen y estadísticas del centro: El menú incluye una opción para mostrar estadísticas del instituto, como la cantidad de ciclos formativos, alumnos por ciclo y total de alumnos, además de una opción para salir.
